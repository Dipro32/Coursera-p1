Welcome to our site

In this site we will list some commonly used commands in github webpage update,status check and to commit local files to remote server

  • To clone your respo into local machine use (git clone your respo link)
  • To check status of the page use (git status)
  • To add your file to your respo use (git add '.')
  • To commit your file use (git commit -m "your message")
  • To send that file to your server use (git push)
